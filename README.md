## 🛸 LAUNCH SEQUENCE

To initialize this system on your local node:

1. **Clone the Uplink**:
   ```bash
   git clone https://github.com/francimolli/portfolio.git
   ```

2. **Refuel Dependencies**:
   ```bash
   pnpm install
   ```

3. **Ignition**:
   ```bash
   pnpm dev
   ```

4. **Access the Interface**:
   Open `http://localhost:3000` to begin the onboarding sequence.
